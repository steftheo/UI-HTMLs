Drunken Parrot UI Kit Getting Started

Getting Started

Drunken Parrot UI Kit is built on top of the popular CSS framework Bootstrap 3. To get started you will need to include both the bootstrap css file which is included with the files under the folder bootstrap, and our Drunken Parrot UI css file in the head of your file.

You will have all the power of the bootstrap framework and the addition of our styles and components.


<!-- Loading Bootstrap -->
<link href="bootstrap/css/bootstrap.css" rel="stylesheet">

<!-- Loading Drunken Parrot UI -->
<link href="css/drunken-parrot.css" rel="stylesheet">

You can then look through the demos and follow through the documentation to see how to use our components. Nearly all our components use the same classes and structure as bootstrap, so its nice and easy to intergrate the styles into your new and existing projects. The documentation will clearly outline how to use any of the new components or variations to existing bootstrap ones.



Javascript

A few of our components require javascript to function correctly such as the checkboxes and radios. The documentation will clearly outline any components that require javascript and how to use them.

We use the jQuery library which can easily be included via any CDN (such as Googles), you may also want to include the bootstrap javascript library if you wish to use any of the bootstrap javascript components.

<!-- Loading Jquery -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>

<!-- Loading Bootstrap Javasvript -->
<script src="bootstrap/js/bootstrap.min.js"></script>


Font Choices

The Drunken Parrot UI comes with two font choices, the default is the sans-serif font Source Sans Pro, we also offer the serif font Merriweather.

To switch the font to Merriweather simply replace the css file drunken-parrot.css with drunken-parrot-merriweather.css. Easy!


<!-- Loading Drunken Parrot UI Merriweather Version -->
<link href="css/drunken-parrot-merriweather.css" rel="stylesheet">


Icons

A great feature of the Drunken Parrot UI kit is the inclusion of the SS Junior icon font. You can find the documentation for the SS Junior icons in the folder /fonts/ss-junior. You dont need to do anything additional to include the icon font, this is all taken care of in the css drunken-parrot.css file.



LESS

We have included the LESS source files for anyone who want to extend or change the source if they want to. All the .less files are found under the less folder and are broken down into modules and other key files such as variables, mixins. The key file is drunken-parrot.less that aggregates the modules and other files into the drunken-parrot.css file.
